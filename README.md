# Biblioteka_BazyDanych
Biblioteka na przedmiot Bazy Danych stworzona z wykorzystaniem MSSQL oraz ASP.NET MVC

## Wykorzystanie technologie
- ASP.NET MVC 5.2.7
- .NET Framework 4.7.2
- Baza danych MSSQL Server 2017

## Pakiety Nu-Get
- EntityFramework 6.2.0
- PagedList 1.17.0
- PagedList.Mvc 4.5.0
